Projektbeschreibung

  • Migration von TnT Dial zu Steam-connect
  • Entwicklung eines Tools zur Datenübertragung
  • Automatisierung manueller Prozesse

Ausgangssituation

Alte System soll ausgeschaltet werden

Kunden ihre Lead Daten müssen migriert werden

Ziele fürs Projekt

  • Mit beide API's erfolgreich kommunizieren
  • Daten format verwandeln (XML -> JSON)
  • Daten strucktur umwandeln (Lead -> Record)

Planung und Umsetzung

  • Wöchenliche cath-ups
  • C# MVC Framework
  • Hilfsmittel
  • Resourcen verteilung

Wirtschaftlichkeitsanalyse

  1. Ausgangslage (Ist-Zustand)
    • Das alte System TnT-Dial ist veraltet und wird abgeschaltet
    • Ohne Migration verlieren Kunden ihre Daten
  2. Problemstellung
    • Migration manuell = hoher Aufwand, fehleranfällig teuer (Stundensätze)
    • Fehlende Automatisierung = mehr Personal = höhere Kosten
  3. Ziel
    • Automatisierte, sichere Migration → spart Zeit, Geld, Ressourcen.
